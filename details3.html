Career Navigator Project

Project Overview

Career Navigator is a platform designed to help users explore career opportunities, connect with mentors, and navigate through various career resources. The site allows users to register, manage their profile, and explore different career paths. With a user-friendly interface, this site provides valuable resources for both career seekers and professionals.

Features

  • User Authentication: Secure registration and login using email or Google authentication.
  • Profile Management: Users can update their profiles, including their name, photo, and other personal information.
  • Career Resources: Access to detailed career guidance, job opportunities, and career-related articles.
  • User Dashboard: Personalized dashboard for registered users with career insights and updates.
  • Contact Us: Easy-to-use contact form for users to ask questions or provide feedback.

Technologies Used

React.js, Firebase, TailwindCSS, React Router, GitHub
