CareMeds Project

Project Overview

CareMeds is a multi-vendor e-commerce platform that specializes in selling medicines and healthcare products. Users can seamlessly purchase medicines, submit queries, and get expert advice. This project demonstrates user and seller management, dynamic data rendering, secure payment processing, and role-based authentication.

Features

  • Interactive homepage with categories and product sliders.
  • User authentication with email/password, Google, and GitHub login.
  • Shop page with filters and sorting for medicines.
  • Secure payment integration using Stripe and invoice generation.
  • Admin dashboard for managing users, products, and payments.
  • Seller dashboard for managing product listings and viewing payment history.

Technologies Used

Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
Authentication: Firebase, JWT
Payment: Stripe
Hosting & Deployment: Vercel, GitHub

Packages Used

axios: For making HTTP requests.
react-query: For managing API state.
sweetalert2: For custom alert messages.
react-icons: For including icons in the UI.
stripe: For secure payment processing.
dotenv: For managing environment variables securely.
